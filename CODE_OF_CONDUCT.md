# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inclusive environment for everyone who wants to contribute to or use this framework, regardless of background, identity, or experience level.

## Expected Behaviour

- **Be respectful** — Treat all participants with respect and consideration
- **Be constructive** — Focus on what's best for the community and the framework
- **Be collaborative** — Work together to improve the content
- **Be professional** — Keep discussions focused on technical and practical matters
- **Accept feedback gracefully** — We're all learning

## Scope

This code of conduct applies to:
- GitHub issues and pull requests
- Discussions in any associated forums or channels
- Any other spaces where this project is discussed

## Reporting Issues

If you experience or witness unacceptable behaviour, please report it by opening a GitHub issue or contacting the maintainers directly.

## Enforcement

Maintainers have the right to remove, edit, or reject comments, commits, code, issues, and other contributions that don't align with this Code of Conduct.
